# chariot

Chariot is a aggregated package manager, currently supports npm, and bower.

# Installation

Place `chariot` in your `bin` folder of choice.

**In OSX** 

```
/usr/local/bin
```

# Usage

```
$ chariot -h
```

**Bower / Web Packages**

```shell
$ chariot web -h
```

**NPM / Node Packages**

```shell
$ chariot node -h
```
